### 브랜드 아키텍처 (2 Layer)

**Layer 1 — 감성 컨셉: "습관"**
- 존재 방식: 매일 하고 싶은 반복 경험
- 태도: 과장 없이 체감으로 증명
- 여정: Stop → Meet → Habit → Change
- 관계: 매일의 홈케어 파트너

**Layer 2 — 제품 컨셉: "Real Difference"**
- 제품 철학: 진짜 변화를 만드는 것만 담는다
- 경쟁 전략: 성분 · 사용감 · 루틴 편의성의 균형
- 원칙: 체감 가능하고 검증 가능한 변화만 약속
- 검증: 데이터 · 사용자 리뷰 · 숫자

### 메시지 체계 (4 Level)

| Level | 메시지 |
|-------|--------|
| L1 Essence | "매일 달라지는 나를 만나다" |
| L2 Promise | "실제로 달라지는 것만 약속합니다" |
| L3 Product | "집에서, 매일, 진짜 달라지는 홈케어" |
| L4 Routine | "하루 5분, 나를 바꾸는 습관" |

### 습관 루프 모델 (Cue → Action → Reward ↻)

- **Cue**: 신호 (세안 완료, 건조 감지, 취침 준비)
- **Action**: 행동 (아침 루틴, 낮 미스트, 저녁 풀케어)
- **Reward**: 긍정 감각 (상쾌한 시작, 촉촉한 감각, 다음날 피부 변화)

## 2. 문제 인식: "습관" 키워드의 한계

기존 메인 키워드 **"습관"**은:
- 브랜드 전반의 감성 앵커로는 적합하나
- **두피 앰플 용기/패키지 디자인을 이끌기엔 너무 포괄적**
- 소비자가 제품을 봤을 때 즉각적으로 느껴야 할 감각을 담지 못함
- 더 구체적이고 시각적으로 연결되는 키워드가 필요

## 4. "얼음공주" 키워드 체계 구체화

### 4-1. 컨셉 정의

**"얼음공주"** = 소비자가 도달하고 싶은 이상적 자기 이미지

> 탄력 있고 주름 없는, 차갑고 탄탄한 피부.
> 과장 없이 전문성으로 증명하는 자신감.
> 프리미엄이지만 매일의 루틴 안에 자연스럽게 녹아드는 케어.

### 4-3. "얼음공주" 서브 키워드 3축

#### 축 1: 전문성 (Professional)

| 요소 | 내용 | 인터뷰 근거 |
|------|------|------------|
| IoT 기술력 | 해비어트 디바이스의 피부 분석 + AI 맞춤 레시피 | "AI 컨설팅 = PMF 검증된 니즈" |
| 권위자 연결 | 의사/약사/AI 전문가 콘텐츠 | "전문가가 맞다면 쭉 쓴다" |
| 데이터 기반 | 수분도/건조도 수치 측정 → 맞춤 케어 | "수치값으로 내 피부 상태를 알고 싶다" |
| 피부과 대안 | 시술과 홈케어 사이의 '중간 단계' | "시술까지는 아직 엄두가 안 나는데" |

→ **디자인 반영**: 정밀한 엣지, 기술적 타이포(DM Sans), 메탈릭 마감

#### 축 2: 냉감 탄력 (Cold Firm)

| 요소 | 내용 | 인터뷰 근거 |
|------|------|------------|
| 탄력/리프팅 | 턱라인 회복, 피부 탄탄함 | "탄력 저하/턱라인 흐려짐이 핵심 고민" |
| 주름 개선 | 진피층 침투의 한계를 디바이스로 극복 | "기초템으로 주름 펴주는 건 안 믿어요" |
| 냉감 경험 | 차갑고 시원한 사용 감각 | 스키니피케이션 + 두피 열감 해소 |
| 스키니피케이션 | "두피가 젊어야 피부가 젊어진다" | 인터뷰 + 업계 트렌드 동시 검증 |

→ **디자인 반영**: 서리 낀 질감, 실버-블루 그라데이션, 투명/반투명 소재

#### 축 3: 시간절약 (Effortless)

| 요소 | 내용 | 인터뷰 근거 |
|------|------|------------|
| 5분 루틴 | 기존 습관에 자연스럽게 편입 | "별도 스텝 추가 없는 통합 케어" |
| 습관화 | 하루 5분, 나를 바꾸는 습관 | "두피에 시간 투자하는 것에 거부감" |
| 간편 설계 | 샴푸와 통합된 케어 | "두피 앰플 안 씀. 바빠서 시간 잡고 해야 하는 게 부담" |

→ **디자인 반영**: 원핸드 드로퍼, 직관적 용기 형태, 미니멀 패키지

## 참고 사례: 루니엘 브랜드의 "곁"

루니엘(RUNIEL)이라는 브랜드는 핵심 키워드로 **"곁"**을 사용해.
"곁"이 성공적인 이유:
- **1음절**, 누구나 아는 일상어
- 즉시 감정이 떠오름 ("곁에 있다" = 안심, 신뢰)
- 브랜드 여정과 연결: 불안 → 곁 → 안착 → 보통의 일상
- 디자인, 카피, CS 정책까지 모든 곳에 적용 가능
- 톤: "차분한 확신" — 조용하지만 확실한

키소나의 새 키워드도 이런 수준의 **직관성과 확장성**을 가져야 해.

## 기존 작업: "얼음공주" 컨셉의 문제

작업 중 임시로 **"얼음공주"**라는 키워드를 사용했어.
이 키워드가 담고 있는 의미:
- 소비자가 도달하고 싶은 이상적 자기 이미지
- 탄력 있고 주름 없는, 차갑고 탄탄한 피부
- 전문성으로 증명하는 자신감
- 메탈릭 실버 + 뮤트 블루의 시각적 연결

**"얼음공주"의 문제점:**
- 4음절, 비유적 → 직관적이지 않음
- 동화적 뉘앙스 → "담백한 확신" 톤과 불일치
- 설명이 필요한 단어 → 브랜드 앵커로 부적합
- 디자인 방향은 맞지만 단어 자체가 브랜드에 안 어울림

**"얼음공주"가 담고 있던 서브 키워드 3축:**
- 전문성 (Professional): IoT/AI 기술, 권위자 신뢰, 데이터 기반
- 냉감 탄력 (Cold Firm): 탄력/리프팅, 주름 개선, 냉감 경험, 스키니피케이션
- 시간절약 (Effortless): 5분 루틴, 습관화, 간편 설계

**디자인 방향 (이 방향은 유지):**
- 용기: 프로스티드 글라스, 실버-블루 그라데이션, 브러시드 실버 캡
- 패키지: 메탈릭 마감, 크리스탈/빙하 패턴
- 컬러: Muted Blue #5e81ac + Silver #c8cfd8
