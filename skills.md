### Brand Core

| 구분 | 내용 |
|------|------|
| 브랜드명 | KISONA |
| **Brand Essence** | **해방 (Liberation)** |
| Brand Statement | **"신경 끄세요."** |
| Sub-message | "복잡한 고민과 정교한 관리는 KISONA가 완벽히 설계했습니다." |

| Brand Persona | 압도적 실력의 외과의사/엔지니어 |
| Brand Journey | 불안 → 만남 → 확신 → 해방 |

### 3대 핵심 철학

| 철학 | 역할 | 설명 |
|------|------|------|
| **선 (The Line)** | 해방을 만드는 설계 | 정밀한 토출구, 흡수 제형, 쿨링 금속 온도 곡선 |
| **좋은 습관** | 해방으로 이끄는 부드러운 루틴 | 하루 5분, 무의식적으로 스며드는 사용감 |
| **No Shortcuts** | 해방된 기다림 (내부 원칙) | 지름길 없이 정직하게, 부담 없는 매일의 관리 |

### Concept Architecture (3 Layer)

| Layer | 키워드 | 역할 |
|-------|--------|------|
| 1. Emotional Essence | **해방** | 소비자가 느끼는 감정적 목적지 |
| 2. Design Language | **선** (The Line) | 해방을 만드는 설계 방법론 (DATA/CONTOUR/ROUTINE LINE) |
| 3. Internal Truth | **No Shortcuts** | 지름길 없이 정직하게 만든다는 내부 원칙 |

### Message Hierarchy (5 Level)

| Level | Message | 용도 |
|-------|---------|------|
| L1 Essence | **신경 끄세요.** | 슬로건, 히어로 |
| L2 Promise | 좋은 습관이 당신을 해방시킨다. | About, 브랜드 소개 |
| L3 Product | 복잡한 고민은 우리가 했습니다. | 제품 카테고리 |
| L4 Routine | 한 번의 루틴이 하나의 불안을 지운다. | 캠페인 |
| L5 Sensory | 쿨링이 스며들고, 두피가 단단해진다. | 두피라인 전용 |

### Liberation Loop

```
불안 인지 → KISONA 루틴 → 하나의 불안 소거 ↻
```

## 2. 에센스 전환 과정

### 2-1. "습관"에서 "해방"으로의 진화 경로

```
습관 → 선(The Line) → 얼음공주 → 밀도 → 선(재탐색) → 소비자 감정 기반 → 해방
```

### 2-2. 핵심 전환 포인트

1. **"선" 컨셉 발견**: 기존 `online_viewer_net (2).htm` 파일에 DATA LINE / CONTOUR LINE / ROUTINE LINE 3축 체계가 이미 존재
2. **소비자 반응 데이터 활용**: 두피 앰플에 대한 8가지 소비자 불안(뾰루지, 떡짐, 효과 없음, 흘러내림 등) 분석
3. **"습관"의 역할 재정의**: 에센스에서 메커니즘("좋은 습관")으로 위치 전환
4. **Softly 연결**: 모회사 "좋은 습관을 만드는 기업" → "좋은 습관이 당신을 해방시킨다"
5. **"선"의 역할 재정의**: 소비자 대면 키워드에서 내부 설계 언어(Design Language)로 전환

### 2-3. 소비자 불안 8가지 (두피 앰플)

| # | 불안 | 과학적 근거 |
|---|------|-----------|
| 1 | 뾰루지 발생 | 두피는 모낭 밀도 높아 모낭염 취약 |
| 2 | 떡짐/다시 감아야 함 | 실리콘·오일 기반 제형이 모발 코팅 |
| 3 | 효과 없음 | 진피 침투 한계, 각질층 잔류 |
| 4 | 앰플 흘러내림 | 드로퍼 토출량 불균일 |
| 5 | 아침 떡짐 | 밤 도포 후 베개 전이 |
| 6 | 여름 기름짐 | 유분계 제형 + 피지 과다 |
| 7 | 자극/알레르기 | pH 불균형, 향료 자극 |
| 8 | 롤온 위생 | 볼 어플리케이터 세균 번식 |

## 4. 비주얼 아이덴티티

### 컬러 시스템 (Clinical Palette)

| 색상 | HEX | 용도 |
|------|-----|------|
| Muted Blue | #5E81AC | 이성과 신뢰의 쿨링 (Point, 20%) |
| Slate Gray | #1E293B | 우아한 무게감 (Base, 70%) |
| Metallic Silver | #C8CFD8 | 프로페셔널 텍스처 (Silver) |
| Red | #F43F5E | 할인, 경고 (Accent, 10%) |

### 타이포그래피 (3 서체)

| 서체 | 용도 | 느낌 |
|------|------|------|
| Pretendard Variable | 한국어/본문 UI | 따뜻하고 현대적 |
| DM Sans | 영문/데이터/제품명 | 기술적, 전문적 |
| Noto Sans KR (Thin/Light) | 디스플레이/브랜드 모먼트 | 가볍고 프리미엄 |

### 버벌 아이덴티티 — "담백한 확신" 톤

| 축 | 위치 |
|----|------|
| 격식 ←→ 캐주얼 | 45% |
| 감성적 ←→ 이성적 | 50% |
| 자극적 ←→ 절제 | 60% |
| 화려한 ←→ 담백한 | 65% |

**DO**: 매일, 체감, 해방, 설계, 루틴, 구체적인 숫자, 직접, 실제
**DON'T**: 기적, 마법, 즉각적, 단번에, 검증 불가한 과장
