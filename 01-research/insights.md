## 1. 핵심 인사이트 9가지

### 인사이트 종합표

| # | 인사이트 | 키소나 액션 |
|---|---------|------------|
| 1 | 기초템 만족 + **디바이스 전환 니즈** | 리프팅/탄력 중심 메시지로 포지셔닝 |
| 2 | **20만원대 = Sweet Spot** | 10만원대는 품질 불신, 30만원대는 부담 |
| 3 | **AI 컨설팅 = PMF 검증된 니즈** | HYAAH '전문가 컨설팅'을 AI로 구현 |
| 4 | **피부과 시술의 중간 단계** | "시술까지는 아직..." 사용자의 홈케어 대안 |
| 5 | **권위자 신뢰 기반 의사결정** | 전문가(약사/피부과) 협업 콘텐츠가 전환 핵심 |
| 6 | 쿠팡 + 네이버 필수 | 회원가입 허들 최소화 |
| 7 | **영상 리뷰가 핵심 결정 요인** | 유튜브 숏폼/롱폼 + 낮은 평점 리뷰 관리 |
| 8 | **두피는 '시간 최소화' 설계** | 별도 스텝 추가 없는 통합 케어 |
| 9 | **습관 변화 스토리텔링** | 제품이 아닌 습관을 파는 브랜드 |

---

### 인사이트 상세

**1. 기초템 한계 인식 → 디바이스 전환 니즈**
- "기초템으로 주름을 펴준다거나 하는 건 그냥 그렇게 안 믿어요"
- 30세 진입 → 안티에이징 → 리프팅 디바이스 관심
- 근거: 인터뷰 직접 발언

**2. 전문가 신뢰 기반 의사결정**
- "전문가가 맞다고 하면 쭉 쓰는 타입"
- HYAAH 브랜드 충성도 = 전문가 컨설팅이 핵심 요인
- 키소나의 AI 컨설팅이 이를 대체할 경쟁우위

**3. 핵심 관심 기능: 리프팅/탄력**
- 턱라인이 흐려지는 실제 체감
- 얼굴을 직접 '밀어올려주는' 물리적 효과 기대
- 속건조보다 탄력이 더 강력한 구매 동기

**4. 피부과 시술의 중간 단계 포지셔닝**
- "선택지가 너무 많으니 피곤한 거예요. 엄두를 못 내고 있죠"
- 시술까지는 아직 → 홈케어 디바이스가 중간 단계

**5. AI 컨설팅 = PMF 검증된 니즈**
- 피부 상태 수치 측정 → 맞춤 레시피 자동 추천
- "HYAAH에서 하는 컨설팅이 AI로 들어가는 건가 싶었어요"

**6. 두피 케어는 '시간 최소화'가 핵심**
- "두피에 시간 투자하는 것에 대한 거부감이 큼"
- 별도 스텝 추가 없이 통합된 케어가 필요
- "두피가 젊어야 피부가 젊어진다" → 인지되지만 행동 연결 안 됨

**7. 스키니피케이션 트렌드**
- "잘 나가는 뷰티 회사들이 밀고 있는 슬로건에 올라타야 한다"
- "두피가 젊어야 피부가 젊어진다" + 구체적 스토리텔링 필요

**8. 구매 여정**
- 개인 SNS 알고리즘 → 복수 인플루언서 교차검증 → 유튜브 숏폼/롱폼 → 상세페이지 리뷰 → 구매

**9. 습관 변화 스토리텔링이 핵심**
- HYAAH 충성도 = [간절한 문제] + [전문가] + [실질적 효과] + [타이밍]
- 키소나도 제품이 아닌 '습관 변화'를 파는 브랜드

## 2. HYAAH 벤치마크 분석

### 충성도 공식

```
[간절한 피부 문제] + [전문가 연결] + [실질적 효과] + [타이밍] = 브랜드 충성도
```

### 키소나 대응 전략

| HYAAH 성공 요인 | 키소나 대응 |
|----------------|-----------|
| 전문가 컨설팅 | AI 컨설팅 (Habiot IoT 디바이스) |
| 맞춤 처방 | AI 기반 피부 분석 + 맞춤 레시피 |
| 지속 사용 유도 | 습관 루프 설계 (Cue → Action → Reward) |
| 커뮤니티 충성도 | 데이터 기반 변화 추적 공유 |

## 3. 스키니피케이션 트렌드

> "잘 나가는 뷰티 회사들이 밀고 있는 슬로건 '스키니피케이션 - 두피가 젊어야 피부가 젊어진다' 여기에 올라타야된다. 그 문장에 대한 구체적인 스토리텔링이 필요하다. 마치 맬리언스의 30년 세탁소집 딸 처럼!"

### 키소나 포지셔닝 기회

- 두피 케어 → 피부 개선의 과학적 연결고리
- "두피부터 탄탄하게, 피부가 달라진다" (L3 Product 메시지)
- IoT 디바이스가 측정 데이터로 변화를 증명하는 차별화
