## 2. Psychographics

### 핵심 가치관

- **전문가 신뢰 기반 의사결정**: "전문가가 맞다고 하면 쭉 쓰는 타입"
- **효율 추구**: 시간 투자 대비 효과를 중시
- **기초템 한계 인식**: "기초템으로 주름을 펴준다거나 하는 건 그냥 그렇게 안 믿어요"
- **검증 욕구**: 복수 인플루언서 교차검증, 낮은 평점 리뷰부터 확인

## 3. Behavioral Patterns

### 정보 탐색 여정

```
인스타 알고리즘 노출
  → 저장/분류
    → 복수 인플루언서 교차검증
      → 유튜브 숏폼/롱폼
        → 상세페이지 리뷰 (낮은 평점부터)
          → 구매 결정
```

### 채널별 역할

| 채널 | 역할 |
|------|------|
| 인스타그램 | 발견 — 알고리즘 기반 첫 노출 |
| 유튜브 | 검증 — 숏폼(빠른 확인) + 롱폼(상세 비교) |
| 상세페이지 | 최종 확인 — 낮은 평점 리뷰가 핵심 판단 근거 |
| 쿠팡 | 비회원 → 재구매 시 쿠팡 선호 |
| 네이버 | 첫 구매 → 회원가입 허들 최소화 |

### 구매 동기 우선순위

1. **탄력/리프팅** — 가장 강력한 구매 동기
2. **전문가 추천** — 권위자(약사/피부과/AI) 신뢰
3. **AI 맞춤 컨설팅** — PMF 검증된 니즈
4. **시술 대안** — "피부과까지는 아직..." 홈케어 포지셔닝
5. **시간 효율** — 두피는 별도 스텝 추가 거부감

## 4. 두피 케어 특수 행동 패턴

### 두피 케어 인식

- **인지 O, 행동 X**: "두피가 젊어야 피부가 젊어진다" — 알지만 실천 안 함
- **시간 투자 거부감**: "두피에 시간 투자하는 것에 대한 거부감이 큼"
- **별도 스텝 부담**: "두피 앰플 안 씀. 바빠서 시간 잡고 해야 하는 게 부담"
- **통합 케어 니즈**: 샴푸와 통합된 케어, 5분 이내 완료 설계 필수

### 가격 인식 (디바이스)

| 가격대 | 인식 |
|--------|------|
| ~10만원 | 품질 불신 ("이 가격에 제대로 될까?") |
| **20만원대** | **Sweet Spot** ("기능할 것 같은 가격") |
| 30만원~ | 부담 ("좀 더 알아봐야 할 것 같은") |

## 5. HYAAH 충성도 벤치마크

은빈의 HYAAH 브랜드 충성도 공식:

```
[간절한 피부 문제] + [전문가 연결] + [실질적 효과] + [타이밍] = 브랜드 충성도
```

### 키소나 시사점

- 제품 판매가 아닌 **'피부 습관 변화' 스토리텔링** 필요
- AI 컨설팅이 HYAAH의 '전문가 컨설팅'을 대체할 수 있는 경쟁우위
- "HYAAH에서 하는 컨설팅이 AI로 들어가는 건가 싶었어요" → PMF 신호
