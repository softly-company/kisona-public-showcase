01 — Brand Essence

두피·피부 불안에서 벗어나는 해방

키소나는 두피·피부 케어에서 느끼는 불안을 정밀한 설계로 해결하여, 고객이 더 이상 신경 쓰지 않아도 되는 상태를 만드는 프리미엄 홈케어 브랜드입니다.

02 — Message

메시지 체계

상황에 맞는 레벨의 메시지를 사용합니다.

Level Message Usage
L1. Essence 신경 끄세요. 슬로건, 히어로
L2. Promise 좋은 습관이 당신을 해방시킨다. About, 브랜드 소개
L3. Product 복잡한 고민은 우리가 했습니다. 제품 카테고리
L4. Routine 한 번의 루틴이 하나의 불안을 지운다. 캠페인
L5. Sensory 쿨링이 스며들고, 두피가 단단해진다. 두피라인 전용

고객 의심 → 키소나 응답

"바르면 뾰루지 나지 않을까?"
→
"두피 자극 테스트 완료. 약산성 pH 5.5 설계."
"바르고 나면 떡지지 않아?"
→
"흡수되는 제형. 떡짐 없이 마무리됩니다."
"써봤자 효과 없을 것 같아"
→
"2주 후 데이터로 확인하세요. IoT가 측정합니다."

Key Verbal Patterns

Evidence Type
체감 + 숫자
"2주 후, 두피 유수분 밸런스 32% 개선. 앱에서 직접 확인하세요."
Liberation Type
루틴 + 해방
"떡짐이 걱정되셨죠. 흡수 제형으로 설계했습니다. 신경 끄세요."
Empathy Type
전/후 + 공감
"바를 때마다 걱정이었습니다 → 이제 바르고 잊습니다."
