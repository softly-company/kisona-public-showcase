01 — Brand Essence

바꿈 없는 일상에서 달라지는 나로

키소나는 정체된 일상에 '매일 체감하는 변화'를 만들어주는 홈케어 브랜드입니다. 거창한 약속이 아닌, 습관이 만드는 진짜 달라짐.

Concept Architecture

두 개의 레이어가 키소나의 세계를 구성합니다.

Layer 1 — Emotional Concept
습관
존재 방식매일 반복하고 싶은 경험
태도과장 없이 체감으로 증명
여정멈춤 → 만남 → 습관 → 달라짐
관계매일의 홈케어 파트너
Layer 2 — Product Concept
Real Difference
제품 철학실제로 달라지는 것만 넣는다
경쟁 전략성분 · 사용감 · 루틴 편의성의 균형
원칙체감 가능한 변화만 약속
검증데이터 · 사용자 후기 · 숫자
02 — Message

메시지 체계

상황에 맞는 레벨의 메시지를 사용합니다.

Level Message Usage
L1. Essence 매일 달라지는 나를 만나다 브랜드 슬로건
L2. Promise 실제로 달라지는 것만 약속합니다 About, 브랜드 소개
L3. Product 집에서, 매일, 진짜 달라지는 홈케어 제품 카테고리
L4. Routine 하루 5분, 나를 바꾸는 습관 캠페인

고객 의심 → 키소나 응답

"홈케어로 뭐가 달라져?"
→
"2주 만에 체감. 실제 사용자 93%가 변화를 느꼈습니다."
"꾸준히 못 할 것 같아"
→
"하루 5분. 이것만 하면 됩니다."
"비싼 거 아니야?"
→
"매일 쓰니까, 하루 ₩890원."
03 — Verbal Identity

담백한 확신

키소나의 말투는 조용하지만 확신에 차 있습니다. 과장하지 않되 분명하게.

Key Verbal Patterns

Evidence Type
체감 + 숫자
"2주 후, 모공 30% 축소. 직접 확인해보세요."
Habit Type
루틴 + 쉬움
"하루 5분. 세안 후, 한 번 바르면 끝."
Empathy Type
전/후 + 공감
"달라진 건 제품이 아니라 나였습니다."
