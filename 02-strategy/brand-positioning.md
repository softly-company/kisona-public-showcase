# Brand Positioning

> 키소나는 두피·피부 불안에서 "해방"시키는 프리미엄 홈케어 브랜드입니다.

---

## 1. Positioning Statement

**KISONA**는 **탄력/두피 고민을 가진 30대 여성**을 위한
**IoT/AI 연동 프리미엄 홈케어** 브랜드로,
**"피부과 시술과 기초 홈케어 사이의 중간 단계"**에 포지셔닝합니다.

---

## 2. Competitive Landscape

### 포지셔닝 맵

```
                    전문성 높음
                        │
                        │
        피부과 시술 ●    │    ● KISONA (target)
                        │      IoT + AI + 앰플
                        │
  ───────────────────── ├ ────────────────────
  가격 높음             │              가격 중간
                        │
        ● 고가 디바이스  │    ● 기초 홈케어
        (50만원+)       │      (기초 화장품)
                        │
                    전문성 낮음
```

| 항목 | 릴리이브 | 리필드 | **KISONA** |
|------|---------|--------|-----------|

| 채널 | 올리브영 1위 | 화해 어워드, CES 2026 | 자사몰 + IoT 연동 |

## 3. Value Proposition (5 Pillars)

| Pillar | 소비자 언어 | 기술적 근거 |
|--------|-----------|-----------|
| **AI 맞춤 케어** | "내 피부를 아는 제품" | Habiot IoT 피부 분석 + AI 맞춤 레시피 |
| **시술급 홈케어** | "피부과까지는 아직... 집에서도 될까?" | 디바이스 + 앰플 시너지, 진피층 침투 |
| **5분 루틴** | "바빠서 시간이 없는데" | 설계된 루틴, 통합 케어 |
| **데이터 증명** | "진짜 효과 있어?" | 수분도/건조도 수치 측정, Before/After |
| **불안 해소** | "뾰루지 나면 어쩌지?" | 피부과 전문의 검증, 약산성 설계 |

## 4. Price Strategy

### 가격 심리 구간

| 가격대 | 소비자 인식 | 키소나 전략 |
|--------|-----------|-----------|
| ~10만원 | "이 가격에 제대로 될까?" (불신) | 이 구간 회피 |
| **20~25만원** | **"기능할 것 같은 가격"** (신뢰+접근성) | **전략가 설정** |
| 30만원~ | "좀 더 알아봐야 할 것 같은" (부담) | 프리미엄 번들용 |

### 가치 방정식

```
체감 가치 = (전문가급 효과 + AI 맞춤 + 불안 해소) ÷ 가격
         = (시술 대안 + IoT 분석 + 설계된 루틴) ÷ 20만원대
         = "하루 ₩890원으로 두피·피부 불안에서 해방"
```

## 5. Target Market

### 핵심 타겟

| 항목 | 내용 |
|------|------|
| 인구통계 | 28~38세 여성, 도시 거주 |
| 핵심 고민 | 탄력 저하, 턱라인 흐려짐, 두피-피부 연결 |
| 소비 성향 | 효율 추구, 전문가 신뢰, 20만원대 수용 |
| 행동 패턴 | SNS 발견 → 교차검증 → 유튜브 → 리뷰 → 구매 |

### 스키니피케이션 포지셔닝

> "두피가 젊어야 피부가 젊어진다"

- 두피 케어 → 피부 개선의 과학적 연결고리
- IoT 디바이스가 측정 데이터로 변화를 증명
- 인지는 있으나 행동 연결이 안 되는 갭 → 키소나가 해결
