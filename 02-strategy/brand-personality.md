# Brand Personality

> 키소나의 말투는 조용하지만 확신에 차 있습니다. 과장하지 않되 분명하게.

---

## 1. Voice Principle: "담백한 확신"

**Understated Confidence** — 과장 없이, 설계와 데이터로 증명하는 목소리.

---

## 2. Tone Spectrum

| 축 | 위치 | 해석 |
|----|------|------|
| 격식 ←→ 캐주얼 | **45%** (약간 포멀) | 딱딱하지 않되 가벼이 여기지 않는 |
| 감성적 ←→ 이성적 | **50%** (균형) | 이성과 감성의 균형, 데이터 + 공감 |
| 자극적 ←→ 절제 | **60%** (절제 쪽) | 선정적이지 않은, 절제된 |
| 화려한 ←→ 담백한 | **65%** (담백 쪽) | 화려하지 않은, 담백한 |

---

## 3. Key Verbal Patterns (3종)

### Pattern 1: Evidence Type — 체감 + 숫자

> "2주 후, 모공 30% 축소. 직접 확인해보세요."

- 근거: 데이터, 임상, 사용자 수치
- 용도: 제품 소개, 상세페이지, 리뷰 인용

### Pattern 2: Liberation Type — 루틴 + 해방

> "신경 끄세요. 설계된 루틴이 불안을 지웁니다."

- 근거: 설계된 제품이 불안을 구조적으로 해결
- 용도: 캠페인, SNS, 패키지 카피

### Pattern 3: Empathy Type — 전/후 + 공감

> "달라진 건 제품이 아니라 나였습니다."

- 근거: 사용자 경험, 감정 변화
- 용도: 브랜드 스토리, 후기 콘텐츠

---

## 4. Words We Use

### 사용하는 말 (DO's)

- 매일
- 체감
- 해방
- 설계
- 루틴
- 구체적인 숫자
- 직접, 실제

### 조건부 (Conditional)

| 단어 | 사용 조건 |
|------|----------|
| "놀라운" | 근거와 함께 |
| "진짜" | 데이터 동반 시 |
| "확실한" | 임상 데이터 병행 |

### 쓰지 않는 말 (DON'Ts)

- 기적
- 마법
- 즉각적
- 단번에
- 검증 불가한 과장

---

## 5. 브랜드 아키타입

**The Liberator × The Sage**

- Liberator: 불안으로부터의 해방, 담담한 확신, 선언적 톤
- Sage: 데이터 기반, 증거 제시, 진실 추구
