# Brand Guideline

> KISONA 종합 브랜드 가이드라인 — 에센스부터 적용까지

---

## PART 1. BRAND CORE

### 1.1 Brand Essence: 해방 (Liberation)

소비자가 두피·피부 케어에서 느끼는 불안(뾰루지, 떡짐, 효과 없음)으로부터의 **해방**.
KISONA는 정밀한 설계로 고객이 더 이상 신경 쓰지 않아도 되는 상태를 만든다.

### 1.2 Brand Personality: 담백한 확신

감성적인 위로 대신 압도적인 기술력과 공학적 설계로 **'확실한 결과'**를 담담하게 쥐여주는 단호한 전문가.

| 축 | 위치 | 해석 |
|----|------|------|
| 격식 ←→ 캐주얼 | **45%** | 딱딱하지 않되 가벼이 여기지 않는 |
| 감성적 ←→ 이성적 | **50%** | 이성과 감성의 균형, 데이터 + 공감 |
| 자극적 ←→ 절제 | **60%** | 선정적이지 않은, 절제된 |
| 화려한 ←→ 담백한 | **65%** | 화려하지 않은, 담백한 |

### 1.3 Brand Persona

> "감성적인 위로 대신 압도적인 기술력과 공학적 설계로 '확실한 결과'를 담담하게 쥐여주는 단호한 전문가."

**아키타입**: Liberator × Sage
- **Liberator**: 불안으로부터의 해방, 담담한 확신
- **Sage**: 데이터 기반, 증거 제시, 진실 추구

### 1.4 Brand Slogan

**메인**: "신경 끄세요."

**서브**: "복잡한 고민과 정교한 관리는 KISONA가 완벽히 설계했습니다. 당신은 매일 바르기만 하면 됩니다."

## PART 2. BRAND SYSTEM — 3대 핵심 철학

### 2.1 선 (The Line) — 해방을 만드는 설계

정밀한 토출구, 흡수되는 제형의 경계, 쿨링 금속의 온도 곡선.
제품의 모든 접점에 존재하는 '선'이 소비자의 불안을 구조적으로 제거한다.

| 선 | 역할 | 설계 요소 |
|----|------|----------|
| DATA LINE | 전문성의 선 | IoT 측정, 우상향 데이터 그래프, 용기 눈금 |
| CONTOUR LINE | 탄력의 선 | 쿨링 메탈, 윤곽 리프팅, 모공 수축 |
| ROUTINE LINE | 루틴의 선 | 5분 루틴, 자석 결합, 오차 없는 습관 |

### 2.2 좋은 습관 — 해방으로 이끄는 부드러운 루틴

일상에 무의식적으로 스며드는 사용감. 번거롭지 않은 루틴이 좋은 습관을 형성하고, 그 습관이 결국 해방을 만든다.

> "좋은 습관이 당신을 해방시킨다."

- 하루 5분, 기존 생활 궤도를 이탈하지 않는다
- 뚜껑 → 바름 → 마사지, 3단계로 완성
- 무의식적으로 반복하게 되는 사용감 설계

### 2.3 No Shortcuts — 해방된 기다림

지름길은 없다. 하지만 정직한 설계와 매일의 루틴이 불안 없이 기다릴 수 있는 확신을 준다.

- 정직한 선언: "빠른 효과"를 약속하지 않는다
- 부담 없는 매일의 관리가, 급하지 않은 변화를 만든다
- 소비자에게 직접 말하지 않지만, 제품의 모든 설계에 녹아 있는 내부 원칙

## PART 3. BRAND APPLICATION

### 3.1 Message Hierarchy (5 Level)

| Level | Message | 용도 |
|-------|---------|------|
| L1 Essence | **신경 끄세요.** | 슬로건, 히어로 |
| L2 Promise | 좋은 습관이 당신을 해방시킨다. | About, 브랜드 소개 |
| L3 Product | 복잡한 고민은 우리가 했습니다. | 제품 카테고리 |
| L4 Routine | 한 번의 루틴이 하나의 불안을 지운다. | 캠페인 |
| L5 Sensory | 쿨링이 스며들고, 두피가 단단해진다. | 두피라인 전용 |

### 3.2 Product Line Messages

| 제품 라인 | 불안 | 해방 메시지 |
|----------|------|-----------|
| 두피 앰플 | 떡짐, 뾰루지 | "떡짐·뾰루지에서 해방" |
| 두피 마사지기 | 두피 스트레스 | "두피 스트레스에서 해방" |
| 뷰티 디바이스 | 피부 걱정 | "피부 걱정에서 해방" |

### 3.3 Verbal Identity

#### 쓰는 말 (DO)

- 짧고 명확한 평서문
- 데이터, 수치 기반 근거
- 매일, 루틴, 설계, 해방, 체감
- 구체적인 숫자 (2주, 5분, 93%)

#### 쓰지 않는 말 (DON'T)

- 기적, 마법, 즉각적, 단번에
- 과장된 감정 호소 ("피부가 환호합니다!")
- 검증 불가한 과장
- 느낌표 남용

#### 문장 규칙

- 한 문장에 하나의 메시지
- 마침표로 끝내되 느낌표는 최소화
- "~합니다" 체 기본 (격식 45%)
- 주어를 생략하여 간결하게

### 3.4 Visual & UX Identity

| 요소 | 방향 |
|------|------|
| 선 모티프 | 기하학적 직선, 눈금선, 정밀한 경계 |
| 색상 | 무채색 기반 + Muted Blue 포인트 |
| 질감 | 무광, 프로스티드, 브러시드 메탈 |
| 레이아웃 | 여백 충분, 수직적 실루엣 |
| UX 원칙 | Zero-Instruction (설명서 필요 없는 직관) |
| 패키징 | Easy-open, 한 손 조작, 자석 결합 |

#### Clinical Palette

| 색상 | HEX | 용도 |
|------|-----|------|
| Muted Blue | #5E81AC | 이성과 신뢰의 쿨링 |
| Slate Gray | #1E293B | 우아한 무게감 |
| Metallic Silver | #C8CFD8 | 프로페셔널 텍스처 |

### 3.5 Team Note — RUNIEL vs KISONA

| 항목 | RUNIEL (곁) | KISONA (해방) |
|------|------------|--------------|
| 에센스 | 곁 | 해방 |
| 페르소나 | 따뜻한 테라피스트 | 압도적 실력의 외과의사/엔지니어 |
| 톤 | 부드럽고 따뜻한 | 담백하고 확신에 찬 |
| 약속 | "곁에 있을게요" | "신경 끄세요" |
| 설계 원칙 | 편안함, 감성 | 정밀함, 설계 |
| 관계 | 함께하는 동반자 | 결과를 보장하는 전문가 |
