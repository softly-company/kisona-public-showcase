# Messaging Framework

> 상황에 맞는 레벨의 메시지를 사용합니다. 모든 메시지는 "해방"을 향합니다.

---

## 1. Message Hierarchy (5 Level)

| Level | Message | Usage |
|-------|---------|-------|
| **L1. Essence** | **신경 끄세요.** | 브랜드 슬로건, 히어로 |
| **L2. Promise** | 좋은 습관이 당신을 해방시킨다. | About, 브랜드 소개 |
| **L3. Product** | 복잡한 고민은 우리가 했습니다. | 제품 카테고리 |
| **L4. Routine** | 한 번의 루틴이 하나의 불안을 지운다. | 캠페인 |
| **L5. Sensory** | 쿨링이 스며들고, 두피가 단단해진다. | 두피라인 전용 |

## 2. 고객 불안 → 키소나 응답

| 고객 불안 | 키소나 응답 |
|----------|-----------|
| "앰플 바르면 뾰루지 나지 않을까?" | "두피 자극 테스트 완료. 피부과 전문의 확인." |
| "바르고 나면 떡져서 다시 감아야 해" | "흡수되는 제형. 떡짐 없이 마무리됩니다." |
| "써봤자 효과 없을 것 같아" | "2주 후 데이터로 확인하세요. IoT가 측정합니다." |
| "앰플이 흘러내려서 아까워" | "정밀 토출구 설계. 1회 사용량 정확히 나옵니다." |
| "아침에 바르면 머리가 떡져" | "속건성 제형. 바르고 5분이면 보송합니다." |
| "여름에 두피가 기름지는데 앰플을?" | "쿨링 포뮬러. 여름에 더 좋습니다." |
| "두피에 자극 줄까봐 무서워" | "약산성 pH 5.5. 민감 두피용으로 설계했습니다." |
| "롤온은 좋은데 위생이 걱정돼" | "교체형 어플리케이터. 매번 깨끗합니다." |

## 3. 카피 패턴

### Pattern 1: Evidence Type (체감 + 숫자)

```
"[기간] 후, [구체적 수치]. [결과 확인 방법]."
```

예시: "2주 후, 두피 유수분 밸런스 32% 개선. 앱에서 직접 확인하세요."

### Pattern 2: Liberation Type (루틴 + 해방)

```
"[불안]. [KISONA 해결]. 신경 끄세요."
```

예시: "떡짐이 걱정되셨죠. 흡수 제형으로 설계했습니다. 신경 끄세요."

### Pattern 3: Empathy Type (전/후 + 공감)

```
"[불안했던 과거] → [해방된 현재]. [한 문장 선언]."
```

예시: "바를 때마다 걱정이었습니다 → 이제 바르고 잊습니다. 그게 해방입니다."

## 4. 제품별 메시지 확장

| 제품 라인 | 고객 불안 | L3 메시지 | L5 감각 카피 |
|----------|----------|----------|-------------|
| 두피 앰플 | 떡짐, 뾰루지 | "떡짐·뾰루지에서 해방" | "쿨링이 스며들고, 두피가 단단해진다." |
| 두피 마사지기 | 두피 스트레스, 탈모 불안 | "두피 스트레스에서 해방" | "차가운 금속이 닿는 순간, 두피가 깨어난다." |
| 뷰티 디바이스 | 피부 효과 불확실 | "피부 걱정에서 해방" | "데이터가 채우고, 결과가 증명한다." |

## 5. 카피 가이드라인

### 문장 규칙
- 한 문장에 하나의 메시지
- 숫자는 아라비아 숫자 사용 (2주, 5분, 93%)
- 마침표로 끝내되 느낌표는 최소화
- "~합니다" 체 기본 (격식 45%)

### 쓰는 단어 (DO's)
매일, 체감, 해방, 설계, 루틴, 구체적인 숫자, 직접, 실제

### 조건부
- "놀라운" — 근거와 함께
- "진짜" — 데이터 동반 시
- "확실한" — 임상 데이터 병행

### 쓰지 않는 단어 (DON'Ts)
기적, 마법, 즉각적, 단번에, 검증 불가한 과장
