# Messaging Framework

> 상황에 맞는 레벨의 메시지를 사용합니다.

---

## 1. Message Hierarchy (4 Level)

| Level | Message | Usage |
|-------|---------|-------|
| **L1. Essence** | 매일 달라지는 나를 만나다 | 브랜드 슬로건 |
| **L2. Promise** | 실제로 달라지는 것만 약속합니다 | About, 브랜드 소개 |
| **L3. Product** | 집에서, 매일, 진짜 달라지는 홈케어 | 제품 카테고리 |
| **L4. Routine** | 하루 5분, 나를 바꾸는 습관 | 캠페인 |

---

## 2. 고객 의심 → 키소나 응답

| 고객 의심 | 키소나 응답 |
|-----------|-----------|
| "홈케어로 뭐가 달라져?" | "2주 만에 체감. 실제 사용자 93%가 변화를 느꼈습니다." |
| "꾸준히 못 할 것 같아" | "하루 5분. 이것만 하면 됩니다." |
| "비싼 거 아니야?" | "매일 쓰니까, 하루 ₩890원." |

---

## 3. 카피 패턴

### Pattern 1: Evidence Type (체감 + 숫자)

```
"[기간] 후, [구체적 수치]. 직접 확인해보세요."
```

예시: "2주 후, 모공 30% 축소. 직접 확인해보세요."

### Pattern 2: Habit Type (루틴 + 쉬움)

```
"하루 [시간]. [행동], [한 번] [동작]하면 끝."
```

예시: "하루 5분. 세안 후, 한 번 바르면 끝."

### Pattern 3: Empathy Type (전/후 + 공감)

```
"달라진 건 [대상]이 아니라 [나/당신]이었습니다."
```

예시: "달라진 건 제품이 아니라 나였습니다."

## 4. 두피 라인 메시지 확장

| Level | 기존 | + 두피 라인 |
|-------|------|-----------|
| L1 Essence | 매일 달라지는 나를 만나다 | (동일) |
| L2 Promise | 실제로 달라지는 것만 약속합니다 | (동일) |
| L3 Product | 집에서, 매일, 진짜 달라지는 홈케어 | 두피부터 탄탄하게, 피부가 달라진다 |
| L4 Routine | 하루 5분, 나를 바꾸는 습관 | 하루 5분, 두피 냉감 케어 |
| **L5 Sensory** | — | **차갑게 채우고, 탄탄하게 올린다** |

## 5. 카피 가이드라인

### 문장 규칙
- 한 문장에 하나의 메시지
- 숫자는 아라비아 숫자 사용 (2주, 5분, 93%)
- 마침표로 끝내되 느낌표는 최소화
- "~합니다" 체 기본 (격식 45%)

### 쓰는 단어 (DO's)
매일, 체감, 달라지다, 루틴, 습관, 구체적인 숫자, 직접, 실제

### 조건부
- "놀라운" — 근거와 함께
- "진짜" — 데이터 동반 시
- "확 달라진" — 비교 사진 병행

### 쓰지 않는 단어 (DON'Ts)
기적, 마법, 즉각적, 단번에, 검증 불가한 과장
