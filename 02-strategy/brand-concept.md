# Brand Concept

> 키소나 브랜드 에센스: 정체된 일상에서 달라지는 나로 이끄는 **"습관"**의 브랜드

---

## 1. Brand Essence

### 한 단어: 습관 (Habit)

키소나는 정체된 일상에 '매일 체감하는 변화'를 만들어주는 홈케어 브랜드입니다.
거창한 약속이 아닌, 습관이 만드는 진짜 달라짐.

### Brand Journey (4 Steps)

```
멈춤 → 만남 → 습관 → 달라짐
```

| Step | 단어 | 의미 |
|------|------|------|
| 1 | **멈춤** | 쓰고 있는데 달라지는 게 없다 |
| 2 | **만남** | 매일 쓰고 싶게 만드는 제품 |
| 3 | **습관** | 매일 하게 된다, 체감이 시작된다 |
| 4 | **달라짐** | 거울을 보면 다르다, 진짜 달라졌다 |

### 일상의 리듬

| 시간대 | 루틴 | 감정 | 카피 |
|--------|------|------|------|
| 아침 | 아침 루틴 | 시작 | "오늘도 한 번 더." |
| 저녁 | 저녁 루틴 | 케어 | "하루의 마무리를 피부로." |
| 매일 | 매일의 변화 | 체감 | "어제보다 오늘이 다르다." |

## 2. Concept Architecture (2 Layer)

### Layer 1 — Emotional Concept: "습관"

| 항목 | 내용 |
|------|------|
| 존재 방식 | 매일 반복하고 싶은 경험 |
| 태도 | 과장 없이 체감으로 증명 |
| 여정 | 멈춤 → 만남 → 습관 → 달라짐 |
| 관계 | 매일의 홈케어 파트너 |

### Layer 2 — Product Concept: "Real Difference"

| 항목 | 내용 |
|------|------|
| 제품 철학 | 실제로 달라지는 것만 넣는다 |
| 경쟁 전략 | 성분 · 사용감 · 루틴 편의성의 균형 |
| 원칙 | 체감 가능한 변화만 약속 |
| 검증 | 데이터 · 사용자 후기 · 숫자 |

---

## 3. Habit Connection

### Habit Loop Design

```
Cue (신호) → Action (행동) → Reward (보상) ↻
```

- **Cue**: 세안 완료, 건조 감지, 취침 준비
- **Action**: 아침 루틴, 낮 미스트, 저녁 풀케어
- **Reward**: 상쾌한 시작, 촉촉한 감각, 다음날 피부 변화

### Daily Rhythm

| 시간 | 신호 | 행동 | 보상 |
|------|------|------|------|
| 07:00 | 아침 세안 후 | 키소나 모닝 루틴 | 산뜻한 시작 |
| 12:00 | 건조함 감지 | 미스트 한 번 | 촉촉한 체감 |
| 22:00 | 잠자리 준비 | 풀 케어 루틴 | 다음 날 아침 달라진 피부 |
